# AWS Lambda Web App Demo

This project demonstrates deploying a simple web application using AWS Lambda Function URL.

## Technologies Used
- AWS Lambda
- Python 3.12
- Function URL (HTTP endpoint)

## Features
- Public web endpoint
- Serverless architecture
- Free Tier compatible
- HTML webpage rendering

## Architecture
User Browser → Lambda Function URL → Lambda → HTML Response
